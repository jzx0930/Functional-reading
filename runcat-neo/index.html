RunCat Neo

macOS 工具選單列用量監控

選單列上會跑步的貓，新版可直接顯示 Claude Code / Codex 的用量。

互動示範

仿 macOS 選單列——貓咪奔跑，旁邊卡片顯示用量。拖滑桿模擬「越忙跑越快」：

🍎 ctx 42% 5h 68% 🐈 14:22

它是什麼

RunCat 是知名的 macOS 選單列小工具（貓會隨 CPU 忙碌加速奔跑）。新版 RunCat Neo 開源，新增「Custom Metrics」外掛，可在選單列即時顯示自訂指標。

對 Claude Code / Codex 使用者的重點

  • 目前用的 model
  • context window 用了多少
  • 5 小時／7 天額度還剩多少

瞄一眼選單列就知道，不用切視窗開網頁。留言補充：context 用量太滿（約 83~84%）時，可能需要壓縮舊內容或開新對話。
