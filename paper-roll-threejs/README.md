# 無限紙帶網頁（Three.js + Fable 5 案例）

- 分類：前端案例 / 創意互動（非工具）
- 一句話用途：一個用 Fable 5 一次生成、沒改半行程式的 Three.js 互動網頁——打開就是一捲永遠印不完的立體設計紙帶，滑鼠移到哪就轉到哪，點一下暫停。
- 來源貼文：Threads [@aistart_up](https://www.threads.com/@aistart_up/post/DbF-A-0mDLG)（8,040 瀏覽）

> 註：這則分享的是**作品／做法**，不是可安裝的工具。

## 作品內容

一條會無限滾動的立體紙帶，有物理感的捲動，邊滾邊在後面印出一張張圖片，整個用 **three.js** 寫成。作者說他只是把在 Pinterest 看到的概念描述給 Claude（Fable 5），一次生成、幾分鐘就拿到能動的版本。

- Demo：Google 搜「**paper roll nine vercel**」第一個
- 原始碼：GitHub 搜作者帳號 **thebuggeddev**

## 值得帶走的重點

流程的變化：以前看到喜歡的設計概念只能收藏，因為「做出來」要好幾天的程式與動畫功夫；現在「看到 → 描述給 AI → 拿到能動的版本 → 再慢慢調」中間那段被壓縮了。靈感的瓶頸從「會不會做」變成「找不找得到值得做的點子」。
