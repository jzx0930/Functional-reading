# Higgsfield MCP

- 官網：https://higgsfield.ai
- 分類：AI 影像／影片生成 / MCP Server
- 一句話用途：透過 MCP 讓 AI agent（如 Claude、Fable 5）調用 Higgsfield 的圖片與影片生成能力，做出高質感視覺與網頁動效。
- 來源貼文：Threads [@dmitriyinin](https://www.threads.com/@dmitriyinin/post/DaxRi-aCQgZ)（7.1 萬瀏覽）

## 它是什麼

Higgsfield 是一個 AI 影像／影片生成平台（GPT Image 2 生圖、Seedance image-to-video 生成鏡頭動態）。Higgsfield MCP 把這些能力接上 MCP，讓支援 MCP 的 coding／creative agent 能在流程中直接呼叫它生成素材。

貼文示範 **Higgsfield MCP + Claude / Fable 5** 產出的高質感視覺網頁效果，引發大量迴響。

## 常見工作流（社群案例）

一種被分享的組合：

- **Claude Code / Fable 5**：負責規劃（storyboard、shot list、每個鏡頭的 prompt、QC）
- **Higgsfield MCP**：影片生成（如 Seedance 2.0 fast）
- 其他：搭配 Suno 等做背景音樂

社群心得：昂貴的模型不一定每個階段都最合適——有人偏好用 Fable 5 做規劃、Opus 4.8 做執行。

## 注意

- 生成需消耗 Higgsfield credits（付費），社群有人反映費用偏高。
- 相關 prompt 有作者放在自家販售頁；此處僅記錄工具本身。
- 這類「一句 prompt 生出漂亮頁面」的展示很多，實用性（響應式、可維護性）仍需自行評估。
