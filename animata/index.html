它是什麼

Animata 是一套開源、免費的 React 動畫元件庫，把「看起來很酷炫、但手刻 CSS 與 JavaScript 會花一堆時間」的動態效果組件化，讓你複製貼上、快速客製。

主要功能

  • 直接複製程式碼即可用，隨插即用
  • 基於 React，深度結合 Tailwind CSS 與 Framer Motion
  • 收錄互動卡片、按鈕特效、動態文字、導覽列等
  • 完全開源免費，適合快速客製
