# Animata

- 連結：https://animata.design
- 分類：前端 UI 元件庫 / 微互動動畫
- 一句話用途：專注於「精緻微互動」的開源動畫元件庫，把酷炫但自己寫 CSS/JS 很費時的動態效果整理成可直接複製使用的 React 元件。
- 來源貼文：Threads [@malilion.dev](https://www.threads.com/@malilion.dev/post/Da_-Cx1iGMr)（1,242 瀏覽）

## 它是什麼

Animata 是一套開源、免費的 React 動畫元件庫，主打提升網站的視覺層次與互動質感。把「看起來很酷炫、但手刻 CSS 與 JavaScript 會花一堆時間」的動態效果組件化，讓你複製貼上、快速客製。

## 主要功能／特色

- **直接複製程式碼即可用**，隨插即用
- 基於 **React**，深度結合 **Tailwind CSS** 與 **Framer Motion**
- 收錄大量創意元件：互動卡片、按鈕特效、動態文字、導覽列等
- 完全開源免費，極適合複製貼上後快速客製化

## 適用情境

Landing Page、個人作品集，或想增加活潑感的創意專案。好的網站體驗常來自使用者與介面互動時那些流暢又驚喜的視覺回饋——Animata 把這些「提升質感但開發成本高」的細節變成易套用的視覺外掛。
