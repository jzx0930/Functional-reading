它是什麼

由知名 TypeScript 教育者 Matt Pocock 維護的 skills 集合，聚焦工程與生產力任務。被 curated-codex-skills 作者推薦搭配使用——其中一個代表性理念是「不要用 plan mode，讓 agent 透過決策樹不斷拷問你、釐清需求」，也成為 fable-method 等工作流的靈感來源。

適用情境

  • 用 Claude Code / Codex / Cursor 做工程與生產力任務
  • 想要一套現成、可靠的 agent 工作流
