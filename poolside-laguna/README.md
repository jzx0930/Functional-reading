# Poolside — Laguna S 2.1（開源寫程式模型）

- 分類：AI 模型 / open-weight / Coding
- 一句話用途：美國新創 Poolside 開源的「寫程式專用」AI 模型，約 118B 參數、open-weight，可自己下載在本機離線執行，資料不外流。
- 來源貼文：Threads [@el_howeed](https://www.threads.com/@el_howeed/post/DbFcrmCIAAQ)（3,152 瀏覽）

## 它是什麼

**Laguna S 2.1** 是 Poolside 推出、主打 coding／agentic 的 AI 模型，重點在於 **open-weight**：你能下載權重、在自己的機器上跑，不必把程式碼交給別人的雲端。約 118B 參數。

意義：過去一年最強的開源 AI 幾乎都來自中國（DeepSeek、Kimi 等），Poolside 這步被視為西方開源想追上來的動作。開源之間越競爭，一般人能免費用到的 AI 就越強越便宜。open-weight 給的自由是：東西在你手上、資料不外流、不被單一公司綁死。

## 誠實的注意事項（作者自己補充，很重要）

- 「打贏更大的對手」主要是 **Poolside 自己公布＋外媒轉述**（自稱西方最強 open-weight、以小博大），第三方獨立跑分還需時間，別當定論。
- 參數量各家報約 118B，屬「約略」數字。
- **open-weight（開放權重、能下載自己跑）≠ 完全開源**（後者連訓練資料都公開）。
- 它主打**寫程式／agentic**，不是全能通用模型。
- 實務門檻：118B 模型要在本機跑，對硬體（GPU／記憶體）要求高，一般舊電腦不一定跑得動。
