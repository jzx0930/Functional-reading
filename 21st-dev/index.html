它是什麼

21st.dev 是社群共建的 React UI 元件庫／市集（官方標語：精心製作的 React 元件與模板，不是 AI slop）。收錄大量高質感介面元件與視覺特效，讓前端與設計工程師直接複製程式碼套用。

主要特色

  • 直接複製程式碼即可用（React、Tailwind CSS、Framer Motion）
  • 無縫接上 shadcn/ui 等生態
  • 元件豐富：按鈕、動態卡片、炫酷背景、微互動
  • 大幅省下手調 CSS 與動畫的時間
