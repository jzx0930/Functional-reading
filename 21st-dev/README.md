# 21st.dev

- 連結：https://21st.dev
- 分類：前端 UI 元件市集 / 設計素材庫
- 一句話用途：一個現代化 UI 元件的開源市集，把「看起來很高級、自己刻很痛苦」的互動介面與特效整理成可直接複製使用的元件。
- 來源貼文：Threads [@malilion.dev](https://www.threads.com/@malilion.dev/post/Da25Zh9E-89)（2,399 瀏覽）

## 它是什麼

21st.dev 是一個社群共建的 React UI 元件庫／市集（官網標語強調是「精心製作的 React 元件與模板，不是 AI slop」）。它收錄大量高質感的介面元件與視覺特效，讓前端與設計工程師直接複製程式碼套用，不用從零手刻複雜的 CSS 與動畫。

## 主要功能／特色

- **直接複製程式碼即可用**：主打 React、Tailwind CSS 與 Framer Motion 技術棧。
- **相容現代前端工作流**：無縫接上 shadcn/ui 等生態。
- **元件類型豐富**：高質感按鈕、動態卡片、炫酷背景（如貼文示範的 Shader Lines）、各種微互動（micro-interaction）元件。
- **省時**：大幅減少手動調校 CSS 與複雜動畫的時間。

## 適用情境

- SaaS 產品的落地頁（Landing Page）
- 個人作品集網站
- 需要驚豔視覺的 Web App

很多時候網站的「高級感」就是靠這些互動細節堆疊出來的；21st.dev 把神人等級的前端動效與排版，變成隨插即用、快速拼裝的靈感素材庫。
