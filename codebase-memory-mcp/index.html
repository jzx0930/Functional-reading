它解決什麼問題

AI 很「健忘」：每次改一點就要把整個專案重讀一遍，慢、燒 token、又容易漏掉關聯。codebase-memory-mcp 先幫專案建索引，存成持久化知識圖譜（函式、類別、呼叫鏈、路由、跨服務連結），AI 用圖查詢就能直接看懂結構。

重點：它本身不含 LLM，只是結構分析後端；智慧交給連上的 MCP agent，因此不需額外 API key。

主要特色

  • 158 種語言（內建 tree-sitter 語法）
  • 大型專案建索引快，宣稱省下大量 token
  • 免費開源、全程本機執行，程式碼不外流
  • 一行指令安裝，支援 Claude Code、Cursor、Codex
