# Icon Animator

- 連結：https://iconanimator.app
- 分類：設計工具 / icon 動畫
- 一句話用途：把靜態 icon 快速做出動態感，讓網站、App 的按鈕、提示、空狀態更有生命力，不用重開大型動畫軟體。
- 來源貼文：Threads [@bing_sunzhi](https://www.threads.com/@bing_sunzhi/post/Da3agntAS6h)（8,515 瀏覽）

## 它解決什麼問題

icon 若只是靜態，有時少了一點引導感；但要自己做 icon 動畫，又常常變成一件麻煩事（得開專業動畫軟體、輸出格式、串接程式）。Icon Animator 專門處理這個「小而常見」的痛點。

## 主要功能／特色

- 把靜態 icon 轉成動態效果
- 適合放在網站、App、Landing Page
- 讓按鈕、提示、空狀態（empty state）等看起來更有生命力
- 對「不想重開大型動畫軟體」的人很友善，門檻低

作者觀點：產品質感很多時候不是靠大功能堆出來，而是這種小地方讓人覺得「這個介面有被照顧到」。

## 使用建議 / 注意事項（含留言區討論）

- 留言有提醒：不必每個常用按鈕都動態化，否則可能增加不必要的閱讀負擔；較好的用法是**平常維持靜態、在「讓使用者知道正在跟哪個元件互動」時才動**。
- 留言區也提到同類／可搭配資源 **LottieFiles**（lottiefiles.com，免費輕量動畫，適合網站與 App）。
