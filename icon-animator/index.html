Icon Animator

設計工具icon 動畫

把靜態 icon 快速做出動態感，讓按鈕、提示、空狀態更有生命力。

互動示範

仿 Icon Animator 概念——把滑鼠移到每個 icon 上看它「動起來」：

愛心跳動
鈴鐺搖擺
星星彈跳
打勾描線

它解決什麼問題

icon 若只是靜態，有時少了引導感；但自己做 icon 動畫又常常很麻煩（要開專業軟體、輸出格式、串程式）。Icon Animator 專門處理這個「小而常見」的痛點。

主要功能

  • 把靜態 icon 轉成動態效果
  • 適合放在網站、App、Landing Page
  • 讓按鈕、提示、空狀態更有生命力
  • 門檻低，不用重開大型動畫軟體

使用建議（留言）：不必每個按鈕都動態化，較好的用法是平常靜態、在「讓使用者知道正在跟哪個元件互動」時才動。可搭配 LottieFiles。
