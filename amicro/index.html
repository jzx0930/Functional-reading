它是什麼

Amicro（Premium React Micro-transitions & Interaction Components）是一套 React 微互動元件庫，把常見但「自己刻很煩」的互動細節整理成即用元件：按鈕 hover、icon 變形、狀態切換、刪除震動、重整旋轉等。

主要特色

  • 適合想讓網站更精緻、又不想從零刻動畫的人
  • 對 landing page、SaaS 產品、AI 工具介面都實用
  • 重點不是炫技，而是靠互動細節堆出「高級感」與良好手感
