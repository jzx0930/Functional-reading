# Amicro

- 連結：https://amicro.vercel.app
- 分類：前端 UI 元件庫 / 微互動（micro-interaction）
- 一句話用途：給 React 專案用的高質感微互動元件庫，把按鈕 hover、icon 變形、狀態切換等細節做成可直接用的元件，讓產品「摸起來」更有質感。
- 來源貼文：Threads [@bing_sunzhi](https://www.threads.com/@bing_sunzhi/post/Da-vwukgTg5)（8,502 瀏覽）

## 它是什麼

Amicro（官網標語：Premium React Micro-transitions & Interaction Components）是一套 React 微互動元件庫。它把常見但「自己刻很煩」的互動細節整理成即用元件，例如：

- 按鈕 hover 效果
- icon 變形
- 狀態切換（state transition）
- 刪除時的震動回饋
- 重新整理時的旋轉動畫

## 主要特色

- 適合想讓網站更精緻、但不想從零刻每個動畫細節的人
- 對 landing page、SaaS 產品、AI 工具介面都實用
- 重點不是炫技，而是靠互動細節堆出「高級感」與良好手感

作者觀點：很多產品差的那一點高級感，通常不是功能不夠，而是互動細節太粗——這類元件庫正好補上。
