# awesome（sindresorhus/awesome）

- 連結：https://github.com/sindresorhus/awesome
- 作者：sindresorhus
- 分類：資源索引 / 開源清單
- 一句話用途：全世界頂級「awesome 精選清單」的終極索引——開源世界的藏寶圖。
- 來源貼文：Threads [@soveryai9319](https://www.threads.com/@soveryai9319/post/Da1PSOwExHf)

## 它是什麼

`awesome` 不是一個軟體，而是一份「清單的清單」：由開源大神 sindresorhus 發起、全球社群共同維護的終極索引，指向各技術領域的精選資源清單。GitHub 星數已達數十萬（貼文稱近 50 萬星）。

範疇極廣：從機器學習、資安攻防、AI Agent 部署方案、Web3 工具、自動化效率腳本，到設計素材、免費書籍，幾乎每個技術領域都有專屬的 awesome 精選清單。

## 特點

- **人工嚴選門檻高**：任何專案要被收錄，需通過嚴格的品質與實用性審查，不是普通的網頁書籤。
- **純共享、無廣告**：走全球社群合力維護的路線，對比那些「把開源工具包一包就收會員費」的資源網站。
- **信息差工具**：順著清單挖冷門的自動化工具，就能在效率上領先同行。

## 使用建議 / 踩坑提醒

- 容易患上「資訊囤積癖」：收藏幾百個清單卻一個都沒點開，流於形式。重點是真的挑幾個來用。
- 內容多為全英文硬核資料，英文不好會較吃力。
- 用法很簡單：加進瀏覽器最愛，需要某領域資源時就進對應的 awesome 清單精準找。
