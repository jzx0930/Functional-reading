背景與理念

提升 agent 表現的三要素：夠好的 prompt、夠完整的 harness、夠全面的 plugins。核心心得：

  1. 把一句 prompt 寫好，效益最顯著、門檻最低。
  2. skills 不是裝越多越好，要裝適合自己的（作者常用不超過 20 個）。
  3. 工程與生產力任務推薦搭配 mattpocock/skills。

內含 skills（v0）

  • prompt-master-gpt5 — 依官方文件把 prompt polish 成「GPT 5.6 最能發揮」的寫法。適用任務已明確、只想優化格式時。
  • prompt-review-and-dispatch — 建立在前者之上，加一層「grilling gate」（讓 agent 透過決策樹拷問你、釐清需求）。適用只知大方向、還不確定具體任務時。

所有輸入採原生「可點選輸入」，不用打字；宣稱已跨平台測試（Windows、Mac、Linux）。目前為 v0。
