Blender MCP

AI × 3DMCP Server

透過 MCP 讓 AI 模型（如 GPT-5.6 Sol、Claude）直接操控 Blender 進行 3D 建模與渲染。

它是什麼

把 Blender 接上 MCP（Model Context Protocol）的伺服器，讓支援 MCP 的 AI agent 能直接下指令操控 Blender：建立／調整模型、佈場景、設鏡頭與材質、觸發渲染等。貼文示範用 GPT-5.6 Sol 調用 Blender MCP 自動完成 3D 場景。

能做 / 目前限制（社群觀點）

看好：真正的專家已把 AI 融入工作流，用它加速粗活。
保留：目前 AI 生成常是「調用現成素材」擺設渲染；在正確拓撲、佈線、UV 拆解、拔模／公差／組裝順序等「可生產」細節仍不足，生成檔案未必能直接用於製造。

結論傾向：AI 是強力輔助，但「只能看不能用」的產出對生產端意義有限；短期較難取代真正掌握 Blender 的人。可對照 GhostPoly AI（主打補上結構化、可生產這塊）。
