# taste-skill

- 連結：https://github.com/Leonxlnx/taste-skill
- 作者：Leonxlnx
- 分類：AI Agent Skill / 前端設計品味
- 一句話用途：一個專門「矯正 AI 前端審美」的開源 skill，讓 AI 寫出的網頁有高級感、擺脫無聊套版感。
- 來源貼文：Threads [@darks0603](https://www.threads.com/@darks0603/post/Day1jKwE3gn)

## 它解決什麼問題

每次叫 AI 寫前端，畫面常常充滿「無聊的套版感」。taste-skill 給 AI 注入一套設計品味規則，矯正它的審美，讓生成的網頁更有「呼吸感」與高級感——你負責把關品味，AI 幫你做粗活。

## 特點

- GitHub 破 6 萬星（依貼文所述）。
- 不需複雜設定，直接載入 skill 即可使用。
- 安裝到 AI Agent 後，會自動為生成的網站注入品味規則，不用重學框架。

## 安裝

用 skills CLI 一行安裝（留言區提供）：

```
npx skills add Leonxlnx/taste-skill
```

之後直接請 agent「做一個漂亮的網頁」即可，skill 會在背景套用品味規則。

## 適用情境

用 AI／Vibe coding 產生前端頁面時，想讓成品少一點 AI 味、多一點設計質感。可搭配其他 coding agent（Claude Code、Codex、Cursor 等）。
