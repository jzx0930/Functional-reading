它解決什麼問題

過去用 AI 生成網頁後，還要另外丟到 Vercel、Netlify 部署。AI Studio 現在能直接把 Vibe coding 做好的網頁一鍵發布上線，減少對其他平台的依賴。

操作步驟

  1. 在 AI Studio 用 Vibe coding 生成完網站，點右上角 Publish。
  2. 選 Custom domain 綁自己的網域，或用 ai-studio.app 免費子網域直接上線。
  3. 按 Deploy，等 1–2 分鐘就能在網路上看到。

延伸：Firebase / Cloud Run（多一步）

  • Firebase Studio / App Builder：Publish → 拿到 firebase.app 網址 → Firebase console → App Hosting → Add custom domain，驗證 DNS 加 A/AAAA。
  • Google AI Studio 網站：通常改用 Cloud Run 部署 → Cloud Run → Custom domains 驗證並設定 DNS。

以上延伸步驟出自貼文中 meta.ai 的回覆，實際綁定情況建議自行驗證。
