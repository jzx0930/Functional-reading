工作流的進化

寫程式→執行→ 操作→讀畫面驗證→修正

給一句「把 App 跑起來，測試整個註冊流程」，Claude 可以：編譯並安裝 App、在模擬器啟動、點擊滑動操作、讀畫面確認結果、發現 Bug 繼續迭代。模擬器就出現在對話旁，開發者也能隨時接手。補上了 AI 寫完程式後的視覺驗證這一塊。
