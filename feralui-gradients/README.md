# FeralUI Gradients

- 連結：https://feralui.dev
- 分類：設計工具 / 漸層產生器
- 一句話用途：一個把「形狀、模糊、噪點、流動效果」都變成可調整參數的進階漸層設計工具。
- 來源貼文：Threads [@bing_sunzhi](https://www.threads.com/@bing_sunzhi/post/DanIjNFAVXd)（16 萬瀏覽）

## 它是什麼

不是普通的雙色漸層產生器。FeralUI Gradients 把漸層當成「視覺實驗」，讓形狀、模糊、噪點（noise）與流動（flow）效果都能直接調整並即時預覽。

## 主要功能

- 內建多種漸層風格可直接套用
- 可調整顏色、柔化（blur）與噪點
- 能加入文字或圖片直接預覽套用效果
- 支援匯出靜態與動態（動畫）效果

## 為什麼好用

貼文作者的觀點：很多時候不是「做不出漸層」，而是「不知道怎麼搭才有質感」。這類工具先把視覺可能性攤開，挑好再放回網站或產品裡，比從零慢慢試快很多——適合拿來快速找靈感、再回頭調成符合品牌與版面的樣子。

## 適用情境

網頁／產品的背景與 Hero 視覺、Banner、簡報底圖等需要有質感漸層的場合。
