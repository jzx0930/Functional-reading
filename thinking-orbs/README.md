# Thinking Orbs

- 連結：https://orbs.jakubantalik.com
- 分類：前端 UI 元件 / AI 介面動效
- 一句話用途：做「AI 正在思考」那種動態球體效果的元件，讓 AI 產品的等待狀態更有情境感、更像「正在處理」。
- 來源貼文：Threads [@bing_sunzhi](https://www.threads.com/@bing_sunzhi/post/DbE7TowAetC)（4,718 瀏覽）

## 它是什麼

Thinking Orbs（Animated thinking orb component）是一個動態球體元件，專門營造「AI 正在思考」的視覺感。它不改變模型能力，但很影響產品給人的感覺與信任感。

## 特色 / 適用情境

- 適合放在 AI 助理、Agent、語音介面
- 比單純的 loading spinner 更有情境感
- 讓等待時間看起來比較像「正在處理」而非卡住
- 對做 SaaS、AI 工具、作品集頁面的人實用

作者觀點：很多 AI 產品差的不是功能，而是互動狀態太冷冰冰——這種小元件補的正是這塊。
